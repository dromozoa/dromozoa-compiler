型強制と型変換

  • マニュアル

ビット演算子は、演算対象がfloatならばintegerに変換する。 ^と/は、演算対象がintegerならばfloatに変換する。 それ以外の算術演算子は、演算対象がintegerとfloatを含む場合、integerをfloatに変換する。

等値演算子は、演算対象がintegerとfloatを含む場合、 floatからintegerへの変換を試行し、変換に成功したらinteger同士で比較する。 そうでなければ、falseを返す。

関係演算子は、演算対象がintegerとfloatを含む場合、 integerの値がfloatで正確に表現できるならば、integerをfloatに変換し、float同士で比較する。 そうでなければ、floorまたはceilを用いる。 結果がintegerで正確に表現できるならば、結果をintegerに変換し、下記の表のようにinteger同士で比較する。 結果がintegerで正確に表現できない場合は、0.0と比較する。 これにより、nan, inf, -infを含むintegerで表現できない値と正しく比較できる。

元となる比較演算 integer同士の比較
i < f i < ceil(f)
i <= f i <= floor(f)
f < i floor(f) < i
f <= i ceil(f) <= i

integerからfloatへの変換

integerの値がfloatで正確に表現できるならば、それが結果になる。 そうでなければ、もっとも近い表現可能な値に変換される（浮動小数点数の丸めモードが影響する？）。 この変換は失敗しない。

floatからintegerへの変換

floatの値がintegerで正確に表現できるならば、つまり、floatの値が整数でintegerの範囲内にあれば、それが結果になる。 そうでなければ、変換は失敗する。
